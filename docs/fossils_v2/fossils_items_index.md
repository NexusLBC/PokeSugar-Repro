# Fossils items index (v2)

Inventaire des IDs d'items contenant `fossil`, consolidé depuis :
- loot tables datapack (`data/**/loot_table/**`),
- mappings machine Cobblemon (`data/cobblemon/fossils/*.json`),
- assets du mod Academy (`mods/academy-...jar`) pour repérer les fossiles spéciaux déclarés côté mod.

> Aucun item fossil trouvé dans des recettes (`data/**/recipes`), des trades (`data/**/trades`) ou des `.mcfunction` de shops dans ce repo.
